制作 OS 的主要流程：
1. 编写 .asm Nasm 汇编文件（后期可能还有 C）；
2. 使用 Nasm 将 .asm 文件编译为二进制文件 .bin；
3. 使用 Cygwin 模拟 Linux 环境将 .bin 转换为 .img 软盘文件；
4. 使用 VMWare 运行测试它。

今天我们不干别的事，就只用让 OS 输出 Hello world，然后停下就好了。\
